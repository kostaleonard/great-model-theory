  14.  def dot(other: NDArray[T])(implicit num: Numeric[T]): Try[NDArray[T]]

    Returns the dot product of this array with another array.

    Returns the dot product of this array with another array.

    If both this and other are 1-D arrays, it is inner product of vectors. If both this and other are 2-D arrays, it is matrix multiplication. If this is an N-D array and other is a 1-D array, it is an inner product over the last axis of this and other; e.g., if this.shape is (m, n) and other.shape is (n,), the result is an NDArray of shape (m,) which is the inner product of all m length n 1D arrays in this with other. If this is an N-D array and other is an M-D array (where M>=2), it is an inner product over the last axis of this and the second-to-last axis of other: dot(this, other)[i, j, k, m] = sum(this[i, j, :] * other[k, :, m]).

    other

    The array to dot.

    num

    An implicit parameter defining a set of numeric operations.
